# Reference Sources

The design intent in this repository is informed by the following public documentation:

- STMicroelectronics STM32F407 datasheet and hardware development guides
- Texas Instruments TPS54331 datasheet and layout recommendations
- Analog Devices LT3652 datasheet and application examples
- Texas Instruments DRV8871 datasheet
- HopeRF RFM95W module datasheet
- Bosch BME280 and BNO055 datasheets

These references are noted for engineering credibility and future refinement; the current repository remains a simplified portfolio project rather than a release-to-manufacturing design package.
